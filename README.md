\# Ore Breaker Incremental



\## 概要

鉱石をクリックして破壊し、資源を集めて破壊力を強化していく

シンプルなインクリメンタルゲーム。



石 → 銅 → 鉄 → アルミ → ダイヤ → ダークマター

という順で鉱石がグレードアップしていく。



\## 開発方針

\- 小さく作ってすぐ動かす

\- 拡張しやすいデータ駆動設計

\- 見た目よりロジック優先



\## 使用技術

\- HTML

\- JavaScript（ES6）

\- フレームワーク不使用

\- 1ファイル構成（index.html）



\## AI（Claude Code）への指示

\- \*\*必ず日本語で回答・コメントを書くこと\*\*

\- まずは最小実装を優先する

\- 不要な抽象化や過剰設計はしない

\- 処理内容が分かるコメントを多めに入れる



\## 実装ルール

\- 状態は `state` オブジェクトで一元管理

\- 鉱石・強化は配列データとして定義

\- DOM操作は関数化する

\- console.log で状態確認できるようにする



\## 現在の仕様（v0.1）

\- クリックで鉱石にダメージ

\- 破壊で資源獲得

\- 破壊力強化のみ実装

\- 累計資源で次の鉱石を解放
