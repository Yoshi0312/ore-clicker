# 鉱石破壊クリッカーゲーム 仕様書（v0.3）

## 1. ゲーム概要

### タイトル（仮）
Ore Breaker Incremental

### ジャンル
クリック型インクリメンタルゲーム（暇つぶし）

### コンセプト
鉱山を探索し、複数の鉱石を同時に破壊。
資源を集めてツール・スキル・仲間を解放し、破壊能力を成長させていく。
**「選択の楽しさ」と「成長の実感」** を重視。

---

## 2. 基本ゲームループ

```
フィールド上の鉱石をクリック
    ↓
ダメージ → 破壊 → 資源獲得
    ↓
資源で強化・スキル解放
    ↓
新しい鉱石種・エリアが解放
    ↓
より効率的な破壊へ
```

### ループの面白さのポイント
- **複数鉱石の選択**: どれを先に壊すか戦略性
- **コンボ/連鎖**: 素早く壊すとボーナス
- **スキル発動**: タイミングを見計らう楽しさ
- **解放の喜び**: 新要素が次々とアンロック

## 4. フィールドシステム（実装済み）

### 概要
画面上に **10×5 = 50個の鉱石** が同時に配置される。
破壊すると新しい鉱石が即座に補充される。

### フィールド仕様

```js
field = {
  rows: 5,              // 行数（固定）
  cols: 10,             // 列数（固定）
  ores: [],             // 現在配置されている鉱石（50個）
  spawnTable: [...],    // 出現テーブル（確率ベース）
}
```

### 鉱石の出現ルール
- 破壊されると即座に新しい鉱石が出現
- 出現する鉱石は **累計資源に基づく確率テーブル** で決定
- 累計資源に応じて上位鉱石の出現率UP
- レア鉱石（銀・金）は特定確率で出現

### 出現確率ロジック（実装版）

```js
// 累計資源に応じて出現鉱石を決定
if (totalResources < 30) return 'stone';
if (totalResources < 100) return random(['stone', 'copper'], [70, 30]);
if (totalResources < 500) return random(['stone', 'copper', 'iron', 'silver'], [40, 35, 20, 5]);
// ... 以降も累計に応じて段階的に解放
```

## 7. コンボシステム（実装済み）

### 仕様
- 鉱石を破壊するとコンボ+1
- **2秒以内** に次を破壊しないとリセット
- コンボ数に応じてダメージボーナス

### コンボボーナス

| コンボ | ボーナス | 表示       |
|--------|----------|------------|
| 1-4    | +0%      | なし       |
| 5-9    | +10%     | COMBO x5!  |
| 10-19  | +25%     | GREAT!     |
| 20-49  | +50%     | EXCELLENT! |
| 50+    | +100%    | FEVER!!!   |

```js
// ダメージ計算時
const comboBonus = Math.min(state.combo * 0.02, 1.0); // 最大+100%
finalDamage = baseDamage * (1 + comboBonus);
```

## 10. 仲間システム（実装済み）

### 概要
資源を使って仲間を雇用。自動で採掘してくれる。
各仲間はDPS（Damage Per Second）を持ち、全鉱石に継続ダメージ。

### 仲間データ構造

```js
{
  id: "apprentice",
  name: "見習い鉱夫",
  emoji: "👷",
  description: "毎秒0.5ダメージを与える",
  baseCost: 100,
  costMultiplier: 1.15,
  dps: 0.5,
  count: 0,
  unlockLevel: 5
}
```

### 仲間一覧（v0.3実装版）

| 仲間名         | 絵文字 | 基礎DPS | 初期コスト | 解放条件   |
|----------------|--------|---------|------------|------------|
| 見習い鉱夫     | 👷     | 0.5     | 100        | Lv5        |
| 採掘ロボ       | 🤖     | 2       | 500        | Lv10       |
| ドワーフ       | 🧙     | 10      | 3000       | Lv15       |

### 自動採掘ロジック

```js
// 100msごとに全仲間のDPSを全鉱石に分散付与
setInterval(() => {
  const totalDPS = calculateTotalDPS(); // 全仲間のDPS合計
  field.ores.forEach(ore => {
    ore.hp -= totalDPS * 0.1; // 100ms = 0.1秒
    if (ore.hp <= 0) destroyOre(ore);
  });
}, 100);
```

### 15.9 オフライン進行 💤

#### 概要

#### 仕様
```js
const offlineTime = (Date.now() - lastSaveTime) / 1000; // 秒
const offlineEarnings = totalDPS * offlineTime * 0.5;  // 50%効率
```

#### 制限
- 最大8時間まで
- 効率50%（オンライン時より減少）
- 仲間・自動採掘のみ有効（スキル無効）

  updateDisplay();
}, 100); // 100msごと
